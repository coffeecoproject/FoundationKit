# Code of Conduct

FoundationKit contributors are expected to keep project discussion professional, technical, and respectful.

## Expected Behavior

1. Focus on the code, design, documentation, and user impact.
2. Give actionable feedback with enough context to reproduce or understand the issue.
3. Respect different experience levels and communication styles.
4. Keep security reports private until maintainers confirm disclosure timing.

## Unacceptable Behavior

1. Harassment, threats, or personal attacks.
2. Publishing private information without explicit permission.
3. Derailing technical discussion with repeated off-topic comments.
4. Intentionally submitting malicious code, credentials, or harmful instructions.

## Reporting

Report conduct issues to the repository maintainers through a private channel. If the issue involves a security concern, follow `SECURITY.md`.

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code.
